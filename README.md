# Machine Learning Project

A comprehensive machine learning project exploring multiple models and techniques.

## Project Structure

```
MiniProjetML/
├── data/
│   └── Titanic-Dataset.csv
└── src/
    └── all_models.py
```

## Contents

- **data/**: Contains the Titanic dataset used for training and evaluation
- **src/**: Python source code for model implementation and evaluation

## Files

- `all_models.py`: Main script containing all machine learning models and analysis
- `Rapport_mini_projet_8_VF.pdf`: Project report (in French)
- `presentation_mini_projet_8_VF.pdf` / `.pptx`: Project presentation slides

## Getting Started

### Prerequisites

- Python 3.x
- Required libraries (see requirements below)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AYM20444/Machine-learning-project.git
cd Machine-learning-project
```

2. Install dependencies:
```bash
pip install pandas scikit-learn numpy matplotlib seaborn
```

### Running the Project

```bash
python MiniProjetML/src/all_models.py
```

## Dataset

The project uses the Titanic dataset containing passenger information including:
- Passenger demographics (age, gender, class)
- Ticket information
- Survival outcomes

## Models

The `all_models.py` script implements and evaluates various machine learning models for the Titanic classification task.
